# Minecraft-1.21.10_3D-Resource_Pack

这是一款免费的3D资源包，通过更换部分Minecraft原版中的方块模型使其更加真实

你只需要点击右侧的 **最新 发行版** 下载**.zip格式的**压缩包**，并将其添加到你的

**/.minecraft/versions/[你的版本名称]/resourcepacks**

内

即可在游戏中 **设置->资源包** 将其**启用**并且**位于原版材质之上**即可生效
