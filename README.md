# Minecraft-1.21.10_3D-Resource_Pack

This is a **free 3D resource package**. By replacing some of the block models in the original Minecraft, it makes the game environment more realistic.

You just need to put it on your 
In the "**/.minecraft/versions/[your version name]/resourcepacks**" directory

You can enable it in the game by going to **Settings -> Resource Pack**, and place it above the original materials. This will take effect.

The currently updated block models:
- Model of the **crafting table**
- Model of the **ladder**
- Handheld model of the **Torch class**
- Model of the **Lantern class**
- Model of the **Chain class**
- Model of the **Planks class**
- Model of the **Log class**
- Model of the **Stripped log class**
- Model of the **Wood class**
- Model of the **Stripped wood class**
